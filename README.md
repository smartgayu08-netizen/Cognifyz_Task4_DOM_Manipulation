# Task 4 - Complex Form Validation and Dynamic DOM Manipulation

## Objective

To extend form validation with advanced rules and implement dynamic updates to the DOM using JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Client-side form validation
* Password strength checker
* Confirm password validation
* Dynamic welcome message
* DOM manipulation using JavaScript
* Client-side routing without page reload
* Responsive user interface
* Navigation between Home, Register, and Contact sections

## Validation Rules

* Name must contain at least 3 characters
* Valid email format required
* Password must contain at least 8 characters
* Password confirmation must match
* Password strength indicator (Weak, Medium, Strong)

## Dynamic DOM Features

* Real-time password strength updates
* Dynamic welcome message after successful registration
* Show and hide sections using client-side routing

## Project Structure

* index.html
* style.css
* script.js

## Learning Outcome

This project helped me understand advanced form validation, DOM manipulation, client-side routing, and interactive web development using JavaScript.
